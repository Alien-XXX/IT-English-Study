# 官方文档词汇库

> 本词汇库专注于 SpringBoot、MySQL、Redis、Java 官方文档中的常见表达

---

## 📂 目录结构说明

### 1. 配置类表达 (Configuration Expressions)
关键词：`Default value is`, `It is recommended to`, `deprecated`, `enabled by default`

### 2. 条件逻辑表达 (Conditional Logic)
关键词：`If set to true`, `Unless otherwise specified`, `When enabled`, `In case of`

### 3. 架构描述表达 (Architecture Description)
关键词：`interacts with`, `routes to`, `is decoupled from`, `is responsible for`

### 4. 参数说明表达 (Parameter Description)
关键词：`Specifies`, `Determines`, `Defines`, `Controls whether`

### 5. 警告与注意事项 (Warnings & Notes)
关键词：`Note that`, `Be aware that`, `Important`, `Caution`

---

每个词汇将包含以下信息：

```

**类型**：配置类/条件类/架构类/参数类/警告类

**文档来源**：SpringBoot / MySQL / Redis / Java

**典型例句**：
> Example sentence from official documentation.

**学习日期**：YYYY-MM-DD
```
