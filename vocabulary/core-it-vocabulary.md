| expiration | 过期 | TTL 过期时间 |

### 3. 常见科技前缀 (Tech Prefixes)
| 前缀 | 含义 | 示例单词 |
|------|------|----------|
| multi- | 多 | multithread, multimedia |
| asynch- | 异步 | asynchronous |
| poly- | 多 | polymorphism |
| mono- | 单 | monolithic |
| micro- | 微 | microservice |
| inter- | 之间 | interact, interface |
| intra- | 内部 | intranet |

### 4. 常见科技后缀 (Tech Suffixes)
| 后缀 | 含义 | 示例单词 |
|------|------|----------|
| -tion | 名词化 | configuration, connection |
| -able | 可...的 | serializable, configurable |
| -er | 执行者 | handler, listener, mapper |
| -ing | 过程/状态 | threading, caching |
