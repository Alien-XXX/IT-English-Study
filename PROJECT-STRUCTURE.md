## 📂 progress/ — 学习进度追踪目录

|------|------|

> 本目录存储分类词汇库，按应用场景组织

| 属性 | 说明 |
|------|------|

| **阅读对象** | 学生 |
| **更新频率** | 高（学习报错日志时更新） |

**词汇分类**：
1. 堆栈追踪词汇 (Stack Trace)
2. 编译时错误 (Compile-time Errors)

**词条格式**：音标、音节拆解、词根词缀、编程类比、常见搭配、典型例句

### `core-it-vocabulary.md`

| 属性 | 说明 |
|------|------|
| **类型** | 词汇库文档 |
| **作用** | 存储后端开发核心 IT 词汇与词根词缀 |
| **阅读对象** | 学生 |
| **更新频率** | 中（专项词汇学习时更新） |

**词汇分类**：
1. 数据库动词 (Database Verbs) — query, insert, truncate, commit...
2. 缓存与内存概念 (Cache & Memory) — eviction, expiration, heap...
3. 常见科技前缀 (Tech Prefixes) — multi-, asynch-, poly-, micro-...

**设计理念**：
- 词根词缀法：理解词源，举一反三
- 领域映射：将词汇与实际开发场景关联

## 📂 phonics/ — 音标与拼读目录

> 本目录存储音标体系和自然拼读规则

### `phonics-guide.md`

| 属性 | 说明 |
|------|------|
| **类型** | 参考资料/学习指南 |
| **作用** | 提供完整的美式音标表和自然拼读规则 |
| **阅读对象** | 学生 |
| **更新频率** | 低（内容相对稳定） |

**核心内容**：
1. **美式音标总表 (48 Sounds)**
   - 元音 (20个)：单元音 + 双元音
   - 辅音 (28个)
   - 每个音标配有例词和类比记忆

2. **自然拼读规则**
   - Magic E 规则（哑巴 E）
   - R 控制元音
   - 常见元音组合
   - 常见辅音组合
   - 常见后缀发音

3. **音节划分规则**
   - VCV / VCCV 模式
   - 编程类比解释

**设计理念**：
- 基于认知科学"语音环路"理论
- 用编程类比帮助记忆（如：后缀 = 注解）

## 📂 sessions/ — 学习会话目录

> 本目录按日期组织每次学习会话的详细记录

### `README.md`

| 属性 | 说明 |
|------|------|
| **类型** | 目录说明 + 模板文档 |
| **作用** | 说明会话记录的命名规则和格式模板 |
| **阅读对象** | AI（用于生成会话记录） |
| **更新频率** | 极低 |

**核心内容**：
- 文件命名规则：`/sessions/YYYY-MM-DD/session-notes.md`
- 会话记录模板（包含：原始文本、生词、拼读规则、语法映射、认知盲区、学习成果）

---

### `YYYY-MM-DD/session-notes.md`（动态生成）

| 属性 | 说明 |
|------|------|
| **类型** | 会话记录文档 |
| **作用** | 记录当日学习的完整过程和成果 |
| **阅读对象** | 学生（用于复习） |
| **更新频率** | 每次学习会话生成一个 |

**会话记录内容**：
1. 学习日期、时长、主题
2. 今日学习的原始文本（报错/文档）
3. 生词记录（音标、词根、含义）
4. 今日拼读规则
5. 语法/逻辑映射
6. 认知盲区识别
7. 学习成果清单
8. 学生自评（用自己的话复述）
9. 下次学习计划

**设计理念**：
- 类似开发日志（Dev Log）
- 支持"回溯"和"知识检索"
- 记录认知盲区，便于针对性强化

## 🎯 使用建议

1. **开始学习前**：阅读 `README.md` 了解项目
2. **每次学习时**：AI 自动创建 `sessions/YYYY-MM-DD/session-notes.md`
3. **学习结束后**：AI 更新 `progress/it-english-tracker.md` 和相关词汇库
4. **复习时**：查阅 `sessions/` 历史记录或 `vocabulary/` 词汇库
5. **练习拼读**：参考 `phonics/phonics-guide.md`
